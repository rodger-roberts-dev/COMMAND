# COMMAND

The Personal command environment for the Command Flagship.

## Purpose

COMMAND is a PowerShell automation environment that displays useful system information and gives control of a Windows machine back to the user.

The goal is to automate repetitive tasks while providing a clean, command-driven interface.

## Current Version

v0.1.0 - Bridge Online

## Current Features

- Startup bridge
- Dynamic greeting
- PowerShell version detection
- Python version detection
- Git version detection
- Awaiting Orders screen

## Philosophy

COMMAND is designed around the idea that the computer should wait for the user's instructions before beginning processes.

The purpose of COMMAND is to return control of the computer to the user, not to Microsoft or to applications that automatically decide what should happen when the computer starts.

The computer should be **Awaiting Orders**.
